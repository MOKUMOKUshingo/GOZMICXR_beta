Add34 change
============
Base: 3d5start54_add_33_xrrighthand_model.zip

Only the AR screen controller mapping was changed.

AR screen control mapping:
- Left stick X/Y: spherical longitude phi / latitude theta
- Right stick Y: screen scale up/down
- Right stick X: parabolic curvature depth
- Left/right triggers: no longer scale the AR screen

Triggers remain available for screen interaction such as play/pause and scrollbar seeking.
All other Add33 settings are preserved.
